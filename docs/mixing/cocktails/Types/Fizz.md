# 菲茲

菲茲與[[Collins|科林]]相似，不同之處是它一般會加入蛋白，一般由烈酒、檸檬汁、糖、蛋白組合而成，最後加入氣泡水。
菲茲類雞尾酒於 19 世紀 70 年代發源於美國。
