# 柯林斯

柯林斯由烈酒、檸檬汁、糖和氣泡水組合而成，一般直接在裝滿冰塊的高球杯中混合製作。
柯林斯酒類起源於 1800 年，由名為約翰•柯林斯的酒保在倫敦發明。
